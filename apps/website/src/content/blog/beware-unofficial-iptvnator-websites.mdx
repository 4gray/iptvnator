IPTVnator is a free, open-source **IPTV player**. It does **not** provide, sell, or
manage IPTV subscriptions, channels, playlists, accounts, or streaming services.
It only opens playlists you already have.

Recently, several websites and "services" have started using the IPTVnator name to
sell IPTV subscriptions or push suspicious downloads. This post clarifies what is
official and how to stay safe.

<Alert type="warning" title="None of these are affiliated with IPTVnator">
  Any site selling "IPTVnator subscriptions", "IPTVnator channels", "premium",
  "activated", or "cracked" IPTVnator, or paid TV packages under the IPTVnator name,
  is **not** connected to this project or its maintainer.
</Alert>

## Official sources — the only ones

- **Landing page & blog:** https://4gray.github.io/iptvnator/
- **Source code:** https://github.com/4gray/iptvnator
- **Downloads (releases):** https://github.com/4gray/iptvnator/releases

If a link is not on this list, treat it as unofficial.

## How to stay safe

Be especially careful if any website asks you to:

- copy and paste commands into your terminal, or run them with admin privileges
- enter your administrator password to "install" the app
- download `.dmg`, `.pkg`, `.exe`, or `.AppImage` files from an unofficial source
- disable your operating system's security warnings
- get a "premium", "cracked", or "activated" IPTVnator build

<Alert type="info" title="A note on terminal commands">
  IPTVnator itself never needs a subscription or an activation step. The only
  terminal command we ever recommend is documented here on the official blog (for
  the macOS "app is damaged" fix). If a command comes from anywhere else, do not run it.
</Alert>

## IPTVnator is not an IPTV provider

There is no official IPTVnator subscription, channel list, streaming account, or
paid support. Anyone selling those under the IPTVnator name does not represent this
project.

If you are ever unsure whether something is official, start from the links above.
